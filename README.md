# Production Calendar

A React + TypeScript production calendar UI for viewing, creating, filtering, searching, and moving production orders across a monthly or weekly calendar timeline.

## Features

- Monthly calendar grid with adjacent-month days
- Weekly view toggle
- Previous month, next month, and This Month navigation
- Production order tags with status-based styling
- Right-side order list with status, duration, and progress
- Create order modal with validation
- Same-area overlap prevention
- Drag and drop order rescheduling
- Local persistence with localStorage
- Status filtering
- Search by label, color code, area, or assignee
- Click-to-highlight from order list to calendar
- Hover fade behavior
- Undo with Ctrl/Cmd + Z
- Toast messages for create, move, and undo

## Tech Stack

- Bun
- React
- TypeScript
- Tailwind CSS v4
- Zustand
- lucide-react

## Getting Started

```bash
bun install
bun run dev
```
